# Flask React MySQL CRUD

A full-stack CRUD application built with Flask, React, and MySQL. The project demonstrates frontend-backend integration, REST-style API communication, and persistent data management through a relational database.

## Overview

This project was developed to demonstrate how a modern web application can connect a Python backend with a JavaScript frontend and a MySQL database. The backend is responsible for handling API requests, business logic, and database operations, while the frontend provides a simple interface for managing records through create, read, update, and delete actions.

## Features

- Create new records
- View existing records
- Update existing records
- Delete records
- Flask backend API
- React frontend interface
- MySQL database integration
- Structured project layout for future extension

## Tech Stack

- Flask
- React
- MySQL
- Python
- JavaScript
